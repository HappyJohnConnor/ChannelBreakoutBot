# ChannelBreakoutBot サンプルコード
## 使い方
configファイルにBitmexのAPIキーを入力後，`python trade.py`で実行．時間足は1分から60分，またはそれ以降まで．`isTest`でBitmexかBitmex Testnetで行うかを判断．

## 戦略
タイトル通り，チャンネルブレイクアウト戦略を用いてます．1時間足18期間が強いらしいです．ドテン売買で行います．
